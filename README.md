# Hello React Rails

> This project is an implementation of connect react-rails gem. Built with Ruby, Javascript and Ruby on Rails

## Built With

- Ruby
- Ruby on Rails
- JavaScript
- CSS

## Getting Started

```
git clone https://github.com/wuyepabdul/hello-rails-backend

cd hello-rails-backend
rails db:create
rails db:migrate
rails db:seed

rails server

```
